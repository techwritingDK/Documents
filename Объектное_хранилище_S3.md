# Объектное хранилище S3
### Простое и надёжное хранение данных в облаке

[Подключить бесплатно на 30 дней →]()

Объектное хранилище — это облачный сервис для хранения любых файлов: документов, медиа, резервных копий, датасетов. Каждый объект доступен по уникальному URL, а работа с ним ведётся через стандартный HTTP API.

Сервис построен на протоколе S3 (Simple Storage Service) — отраслевом стандарте облачного хранения. Любое приложение, которое работает с облачными хранилищами, подключится к нему без дополнительной настройки.

---

## Какие задачи решает хранилище

Хранилище подходит для задач любого масштаба — от личного фотоархива до петабайтных датасетов.

**Бэкапы и восстановление**
Автоматическое резервное копирование баз данных и файлов. Версионирование позволяет откатиться к любой точке. Восстановление займёт минуты, а не дни.

**Раздача контента (CDN)**
Файлы отдаются по HTTP — изображения, видео, JS/CSS. Географически распределённое хранение ускоряет загрузку.

**Статические сайты**
Разместите HTML, CSS, JS и привяжите свой домен — хостинг без отдельного сервера.

**Big Data и ML**
Узлы кластера обмениваются датасетами через хранилище. Поддержка объектов любого размера.

**Артефакты CI/CD**
Бинарники, Docker-образы, дистрибутивы — складывайте в хранилище прямо из пайплайна с версионированием.

**Архив логов и аналитики**
Долгосрочное хранение логов, истории транзакций, event-стримов и снапшотов.

**Создание presigned URL**
Генерируйте временные ссылки на файлы для безопасной передачи без открытия публичного доступа.

**Интеграция с приложениями**
Подключите хранилище к CRM, ERP или собственному сервису за один день. Готовые SDK для популярных языков программирования.

## Классы хранилища

Выбирайте оптимальный вариант хранилища под ваши задачи:

- **Стандартное** — для часто запрашиваемых данных.
- **Холодное** — для архивов и бэкапов, хранение дешевле.
- **Ледяное** — для регуляторных архивов, годовых и финальных бэкапов, данных закрытых проектов, устаревших логов.

## Почему стоит выбрать объектное хранилище S3

### Безопасность

- Данные шифруются при передаче и хранении. Доступ настраивается гибко: для каждого файла или папки отдельно. Ключевые меры включают версионность, Object Lock для защиты от удаления / шифрования и использование HTTPS для защиты данных.
- Соответствие международным стандартам безопасности (SOC, ISO 27001, PCI DSS) и стандартам защиты персональных данных.
- Соответствие 152-ФЗ для обработки персональных данных 3 и 4 уровня защищённости.

### Надёжность

- Каждый файл хранится в нескольких копиях на разных серверах. Потеря данных исключена даже при аппаратном сбое.
- Сохранность данных на уровне 99,999999999% за счёт репликации данных между несколькими независимыми зонами доступности.

### Масштабируемость и доступность

- От одного гигабайта до петабайт — количество объектов и общий объём данных в бакетах без ограничений. Хранилище увеличивается автоматически — даже в момент пиковой нагрузки.
- Хранилище обеспечивает уровень доступности 99,99% согласно соглашению об уровне сервиса (SLA).
