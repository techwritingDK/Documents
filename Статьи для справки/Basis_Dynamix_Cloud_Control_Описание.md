# Описание оркестратора Basis Dynamix Cloud Control

---

## Общее описание

**Basis Dynamix Cloud Control (BDCC)** — программный оркестратор для управления распределённой облачной инфраструктурой из единой панели управления. Он объединяет разрозненные виртуальные дата-центры в общую экосистему для автоматизированного обслуживания.

Панель управления BDCC открывает доступ к основным функциям системы. Это веб-портал самообслуживания, через который пользователи управляют облаком.

---

## 1. Общее описание архитектуры

Пользователи имеют доступ к облачным ресурсам через единую панель управления. Система определяет доступные функции на основе роли пользователя. Например:

- **Пользователь партнёрской организации** управляет клиентами.
- **Пользователи клиентской организации** работают с облачными ресурсами — виртуальными машинами, сетями и другими объектами.

BDCC использует инструменты резервного копирования в зависимости от платформы виртуализации:

- В **OpenStack** система применяет встроенный сервис Cinder Backup.
- В **VMware vSphere** — приложение Veeam Backup & Replication, которое лицензируется отдельно.

---

## 2. Ролевая модель BDCC

В BDCC действуют следующие роли:

1. Администратор платформы
2. Администратор партнёра
3. Администратор клиента
4. Пользователь клиента
5. Специалист технической поддержки клиента

Каждая роль определяет набор прав и зону ответственности. Пользователь управляет только объектами своего уровня и уровня ниже.
